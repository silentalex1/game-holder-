BatProx Engine API

Integrate the industry's most advanced stealth proxy engine directly into your applications. The BatProx API allows developers to build custom proxy solutions, bypass complex firewalls, and manage traffic flow using our proprietary node network.

Quick Start

Initialize the BatProx core engine in your project. No backend required.

const BatNode = new BatProx({ key: "YOUR_PUBLIC_KEY", region: "auto", stealth: true });

Configuration Standard

Basic configuration options available for all users.

BatNode.config({ errorReporting: true, stats: "minimal", debugMode: false });

Performance Optimizer Premium

Unlock enterprise-grade traffic management features.

BatNode.optimizer.engine = { loadBalancing: true, priorityBandwidth: "high", reduceCongestion: true, edgeNodes: ["us-east", "eu-west"], lowerLatency: true };

Discord Integration Standard

Securely route traffic statistics to your Discord bot without exposing tokens client-side. Our relay handles the authentication.

BatNode.request.discord({ botId: "8928...", method: "traffic_alert", payload: { status: "online", users: 142 } });

Traffic Inspector Premium

Real-time packet inspection and modification.

BatNode.traffic.inspect((packet) => { if (packet.type === "blocked") { return BatNode.reroute(packet, "secure-node-7"); } });
